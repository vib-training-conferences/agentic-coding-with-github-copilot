# Session 01: Chat-Based Coding

1. **Writing functions**: Use GitHub Copilot Chat to write Python bioinformatics functions (GC content, DNA transcription, reverse complement) from natural-language descriptions. Learn to verify and correct AI-generated code.

2. **Debugging**: Use Copilot Chat to identify and fix four bugs in a gene expression analysis script — a mix of logic errors, incorrect calculations, and missing edge-case handling.

3. **Optimizing code**: Use Copilot Chat to replace slow Python loops with vectorized pandas/NumPy operations in a differential expression script. Benchmark before and after.

4. **Data analysis**: Use Copilot Chat (or ChatGPT) as a data analysis assistant to explore, visualize, and statistically analyse a gene expression CSV dataset. Critically evaluate Copilot's suggestions.

# Session 02: Agentic Coding

5. **Writing functions (autocomplete & prompting)**: Explore GitHub Copilot's inline autocomplete (comment-triggered, docstring-triggered), inline chat, and observe the stochastic nature of LLM output by repeating the same prompt multiple times.

6. **Adding documentation**: Use Copilot to generate NumPy-style docstrings for a set of undocumented Python functions in a differential expression analysis module.

7. **Translating scripts**: Use Copilot to translate a Python sequence-analysis script to idiomatic R and to C++17. Compare the outputs and correct any errors.

8. **Debugging faulty scripts**: Find and fix seven bugs (mix of runtime errors, logic errors, and edge-case failures) in a differential expression pipeline with Copilot's help.

9. **Refactoring**: Identify code-quality issues (magic numbers, duplication, oversized functions, inefficient pandas operations) in a working analysis script and refactor it with Copilot.

10. **Data analysis pipeline (Jupyter)**: Build a complete gene expression analysis pipeline step by step in a Jupyter notebook, using Copilot's ✨ generate button and inline chat to create each new cell.
